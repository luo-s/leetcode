# 数组的双变量问题

在处理数组中满足某种条件的双变量或多变量问题时，可以引入哈希表，在遍历数组的同时维护遍历数组得到的信息。这样以空间换时间，一次遍历即可解决问题， 避免暴力枚举的双重循环。
